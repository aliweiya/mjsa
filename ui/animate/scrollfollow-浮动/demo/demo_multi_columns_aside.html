These pages are demonstrations and work-helpers of the "scrollfollow" jQuery plugin, constructed to allow DOM content blocks to follow scrolling in part of or the total page.
It requires the jQuery javascript library to work. A specific plugin's debugger is used in these pages, showing the different values of current scrolling and blocks positions.
To implement scrollfollow, just have a look in the HTML source code of these pages or see README file ...
For information, support and source code, see https://github.com/pierowbmstr/jquery-scrollfollow.
