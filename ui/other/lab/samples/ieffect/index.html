jQuery iEffect plugin

介绍

iEffect 的英文全称是“Interactive Effect”，目的用来实现交互时的样式效果变换。目前，支持的交互事件有 mouseover，mousedown，focus 及自定义事件 error 和 normal。需要说明的是，你需要预定义对应于不同事件的样式类，然后设置为 iEffect 的参数。

对于 mouseover，mousedown，focus 等事件，其相应的恢复事件会被自动绑定，所以你不用关心 mouseout，mouseup，blur 等事件。

示例列表如下：

  • 输入框的效果变换
  • 为按钮元素加入按下样式
  • 变换 div 元素的效果

输入框的效果变换

下面的输入框加入了鼠标浮动，焦点、错误状态效果。

代码：

<input id="txt-test" class="txt-normal" type="text">
<div>
<input id="error" type="button" value="change to error state"/>
<input id="normal" type="button" value="restore to normal state"/>
</div>
<script>
(function($) {

$(".txt-normal").iEffect({
	mouseover: "txt-mouseover",
	focus: "txt-focus",
	error: "txt-error"
});

$("#error").click(function() {
	$("#txt-test").trigger("error");
});

$("#normal").click(function() {
	$("#txt-test").trigger("normal");
});

})(jQuery);
</script>

为按钮元素加入按下样式

下面的例子通过变换 input 元素的样式，来实现鼠标按下的效果。

代码：

<input class="btn-accept-normal" type="button" value="确定">
<input class="btn-cancel-normal" type="button" value="取消">
<script>
(function($) {

$(".btn-accept-normal").iEffect({
	mousedown: "btn-accept-down"
});

$(".btn-cancel-normal").iEffect({
	mousedown: "btn-cancel-down"
});

})(jQuery);
</script>

变换 div 元素的效果

我们还可以应用其到 div 或任意其他元素上，实现效果的变换。

代码：

<div class="big-box-normal"><div class="small-box-normal"></div></div>
<script>
(function($) {

$(".big-box-normal").iEffect({
	mouseover: "big-box-hover",
	mousedown: "big-box-down"
});

$(".small-box-normal").iEffect({
	mouseover: "small-box-hover",
	mousedown: "small-box-down"
});

})(jQuery);
</script>
