jQuery box plugin

介绍

jQuery box 插件可以用来方便地获取和设置任意可视网页元素的盒模型相关量。盒模型相关量包括 padding，margin，border，width 和 height。当然，padding，margin，border 还可以与 top，right，bottom，left 四个方向值相互组合，生成十二个具体量。一般情况下，为了获取某个盒模型量值，你需要获取其 css 的样式值再转换成整型值。更进一步，如果获取的宽度或高度还要包括 padding 与 border 部分，则需要将 border-left-width，border-right-width，padding-left，padding-right 的样式整型值与 height 进行累加，或 border-top-width，border-bottom-width，padding-top，padding-bottom 的样式整型值与 width 进行累加。

jQuery box 插件通过自定义的缩写标识可以方便地获取某个元素的盒模型量数值，或几个盒模型量值的累加。另外，还提供了 boxWidth 与 boxHeight 方法，来获取或设置包含 margin，padding 与 border 的整体盒模型宽度或高度值。

box 方法参数说明

box 方法的使用依赖于选择器，其用于获取某个可视元素的盒模型量数值。使用格式为，$(object).box("ih", "pl", ...)，通过传入样式名缩写，即可获得相应的值。下面对缩写值进行说明：

基于高度与宽度的缩写

  • ih 或 iw：内部高度或内部宽度，等价于通过 $(object).height() 或 $(object).width() 获取的值。

  • oh 或 ow：外部高度或外部宽度，其为元素的整体高度和宽度，包括元素的 border，padding，margin 部分。

基于 margin，border，padding 头开的缩写

  • mt，mb，ml，mr，mtb 和 mlr：分别对应 margin-top，margin-bottom，margin-left 和 margin-right，margin-top 加上 margin-bottom 的缩写，margin-left 加上 margin-right 的缩写。

  • bt，bb，bl，br，btb 和 blr：分别对应 border-top-width，border-bottom-width，border-left-width，border-right-width，border-top-width 加上 border-bottom-width 的缩写，border-left-width 加上 border-right-width 的缩写。

  • pt，pb，pl，pr，ptb 和 plr：分别对应 padding-top，padding-bottom，padding-left 和 padding-right，padding-top 加上 padding-bottom 的缩写，padding-left 加上 padding-right 的缩写。

基于 left，right，top，bottom 开头的缩写

  • l，r，t，b：分别对应左、右、上、下四个方向 margin，border，padding 的累加。

  • lr 和 tb：分别对应左右 margin，border，padding 同时累加的缩写，或上下 margin，border，padding 同时累加的缩写，顺序不限。注意，tb 和 bt 不等同，tb 用来取上下，bt 代表上边框的宽度。

box 方法使用示例

200x100, magin: 2px, border: 4px, padding: 10px

代码：

<style type="text/css">
#box1 {
	background: #ccc;

	width: 200px;
	height: 100px;

	margin: 2px;
	border: 4px solid red;
	padding: 10px;
}
</style>
<div id="box1">200x100, magin: 2px, border: 4px, padding: 10px</div>
<input type="button" id="btn1" value='$("#box1").box("ih", "ptb", "btb")' />
<input type="button" id="btn2" value='$("#box1").box("ow", "l", "r")' />
<script>
(function($) {
$(function() {

$("#btn1").click(function() {
	alert($("#box1").box("ih", "ptb", "btb"));
});

$("#btn2").click(function() {
	alert($("#box1").box("ow", "l", "r"));
});

});
})(jQuery);
</script>

boxWidth与 boxHeight 方法使用示例

代码：

<style type="text/css">
#box2 {
	background: #ccc;

	width: 200px;
	height: 100px;

	margin: 2px;
	border: 4px solid red;
	padding: 10px;
}
</style>
<div id="box2"></div>
<input type="button" id="get-box-height" value='get box height' />
<input type="button" id="get-box-width" value='get box width' />
<input type="button" id="set-box-height" value='set box height' />
<input type="button" id="set-box-width" value='set box width' />
<script>
(function($) {
$(function() {

$("#get-box-height").click(function() {
	alert("current boxHeight is: " + $("#box2").boxHeight());
});

$("#set-box-height").click(function() {
	$("#box2").boxHeight(200);
	alert("now, current height is: " + $("#box2").box("ih"));
});

$("#get-box-width").click(function() {
	alert("current boxWidth is: " + $("#box2").boxWidth());
});

$("#set-box-width").click(function() {
	$("#box2").boxWidth(200);
	alert("now, current width is: " + $("#box2").box("iw"));
});

});
})(jQuery);
</script>
