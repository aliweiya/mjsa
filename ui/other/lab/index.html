jQuery Plugin's Samples

  • box：用来方便地获取和设置任意可视网页元素的盒模型相关量。

  • dialog：提供一组易于定制，且可跨浏览器使用的对话框功能，同时具有易用的对话框回调接口。

  • hash：用来做为一种本地临时存储的解决方案，其实现基于 url 的 hash 部分。

  • iButton：完全模拟浏览器的按钮外观及行为，可以将不同浏览器中的按钮样式进行统一。

  • iEffect：用来实现交互时的样式效果变换。

  • iPrompt：用来在输入框内加入文本提示。

  • pubsub：用来在当前页面内创建一个消息中枢。

  • textbox：是对 HTML 的 textare 元素的增强，主要用于最大输入长度的交互限制。

  • viewport：用来提供视口内异步数据或资源加载的通用性支持。
