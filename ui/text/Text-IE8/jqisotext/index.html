JQISOTEXT

Pseudo-isometric text effect

jqIsoText is jQuery plugin for displaying your text with pseudo-isometric effect.

jqIsoText is compatible and fully tested with Safari 2+, Internet Explorer 6+, Firefox 2+, Google Chrome 3+, Opera 9+.


Examples


This sentence will be treated with default options
$('.default').jqIsoText()

For this sentence words will be treated separatly
$('.split').jqIsoText({ split:'yes' })

Use custom sizes and start from bigger
$('.big').jqIsoText({ fromSize: 50, toSize: 10 })
