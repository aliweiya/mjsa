Dropdown Kit

Show and hide HTML elements with jQuery.

  • EN
  • FR

Presentation

The purpose of Dropdown Kit is to gather common dropdown-like jQuery functionalities (such as dropdown list, combobox, toggle menu, tooltip) into one lightweight and flexible plugin. Dropdown Kit is not revolutionary. It only does what it is meant for: hide and show HTML elements.

Documentation

HTML

HTML elements

Anchor

CSS class name: ddkit-anchor.

This is the main container.

Optional CSS class:
  • ddkit-opened: opens the target by default.

Trigger

CSS class name: ddkit-trigger.

The element that triggers the dropdown.

Optional attributes:
  • position: defines the target position.
  • alignment: defines the target alignment.
  • rel: links the trigger to an external target element.

Target

CSS class name: ddkit-target.

This is the animated element (dropdown).

HTML code

Here is a representative HTML structure to be used with Dropdown Kit:

<!-- Anchor -->
<anytag class="ddkit-anchor">

    <!-- Trigger -->
    <anytag class="ddkit-trigger">the trigger label</anytag>

    <!-- Target -->
    <anytag class="ddkit-target">
        the target content (list, image, div, etc.)
    </anytag>

</anytag>

Using HTML attributes and optional CSS class:

<!-- Anchor -->
<anytag class="ddkit-anchor ddkit-opened">

    <!-- Trigger -->
    <anytag class="ddkit-trigger" rel="#ExternalTarget" position="'top'" alignment="2">the trigger label</anytag>

    <!-- Target -->
    <anytag class="ddkit-target">
        the target content (list, image, div, etc.)
    </anytag>

</anytag>

CSS

Core styles

All Dropdown Kit CSS styles can be found in the dropdownkit.css file.

Feel free to copy and adapt it.

In the <head> section of the HTML page:

<!-- Plugin styles -->
<link rel="stylesheet" type="text/css" href="css/dropdownkit.css" media="screen, projection" />

Javascript

Library

The main javascript file is currently jquery.dropdownkit.1.0.js.

You can optionally use jQuery add-ons:

  • jquery.easing.1.3.min.js, for easing effects on transitions. Check the easing plugin homepage.

Usage

In the <head> section of the HTML page:

<!-- jQuery library -->
<script type="text/javascript" src="your_js_path/jquery-1.8.3.min.js"></script>

<!-- jQuery add-ons (optional) -->
<script type="text/javascript" src="your_js_path/jquery.easing.1.3.min.js"></script>

<!-- Dropdown Kit core -->
<script type="text/javascript" src="your_js_path/jquery.dropdownkit.1.0.js"></script>

<!-- Dropdown Kit execution -->
<script type="text/javascript">
    $(window).load(function(){
        
        $(".anytagselector").dropdownkit();

    });
</script>

You can define the parameters of the plugin:

<!-- Dropdown Kit execution -->
<script type="text/javascript">
    $(window).load(function(){
        
        $(".anytagselector").dropdownkit({
            bodyclick:false,
            autoclose:false
        });

    });
</script>
Check the 'Options' section below.

Options

Name Description Default Values
position Position of the target element 'bottom' 'top', 'bottom', 'right', 'left'
alignment Alignment of the target element 0 0, 1 or 2. Here is the nomenclature, depending of the position:
0 1 2
'top' or 'bottom' align left center align right
'left' or 'right' align top center align bottom
ymargin Set a top or bottom margin (depending of the position) between the trigger and the target 0 An integer
xmargin Set a left or right margin (depending of the orientaion) between the trigger and the target 0 An integer
nopos If set to true, the target won't be positioned with Javascript false true, false
hover If set to true, the target will appear on trigger 'mouseover' instead of 'click' false true, false
closemates If set to true, only one target can be visible at a time true true, false
bodyclick If set to true, will hide all targets (or targets of a group) on body click true true, false
autoclose If set to true, will hide the target after a few seconds. See 'wait' option true true, false
closebtn If set to true, generates a close button tag inside the target element: .ddkit-close false true, false
group Links anchors together in a group. Only group members will interact null A string
targetrel Links the trigger element to an external target (outside the anchor container) null A selector. e.g: '.myclonedtarget', '#theclonedtarget'
targetclone Allows to link several anchors to the same target element null A selector. e.g: '.myclonedtarget', '#theclonedtarget'
wait Time in ms before the target is automatically closed 2000 An integer
infx Target showing animation effect. 'fade' 'fade', 'none', 'slidedown' (only for 'bottom' position)
inspeed Animation speed in ms 500 An integer
ineasing Animation easing function name null A proper easing effect name (you can use an external plugin for that)
incallback A callback function to be called at the end of the animation null null, function(){}
outfx Target hiding animation effect. 'none' 'fade', 'none', 'slideup' (only for 'bottom' position)
outspeed Animation speed in ms 200 An integer
outeasing Animation easing function name null A proper easing effect name (you can use an external plugin for that)
outcallback A callback function to be called at the end of the animation null null, function(){}

API Methods

setTargetPos ( position, alignment )

Changes the position/alignment of a target element.

openTarget ()

Opens a target element.

closeTarget ()

Closes a target element.

closeAllTargets ()

Closes all target elements.

Troubleshooting

Common errors

If you are having trouble running the plugin, start by checking these basic points:

  • Make sure the 3 main HTML tags exist in the code (anchor, trigger and target).
  • If you are using external or cloned target, make sure the selector character comes before the target tag name parameter ('.' or '#).
