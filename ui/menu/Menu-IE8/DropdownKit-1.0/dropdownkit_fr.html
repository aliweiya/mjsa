Dropdown Kit

Afficher/Cacher des contenus avec jQuery.

  • EN
  • FR

Présentation

L'objectif de Dropdown Kit est de simplifier l'utilisation d'animations jQuery frontoffice basiques de type 'dropdown' (liste/menu déroulant, élément 'toggle', tooltip, menu avec onglets, etc.) en utilisant un unique plugin souple et léger. Dropdown Kit n'a rien de révolutionnaire. Il fait simplement ce qu'on attend de lui : masquer et afficher des éléments HTML.

Documentation

HTML

Elements HTML

Anchor

Nom CSS : ddkit-anchor.

Le conteneur principal appelé 'ancre'.

Classe CSS (optionnelle) :
  • ddkit-opened : définit l'élément 'target' comme affichée par défaut.

Trigger

Nom CSS : ddkit-trigger.

L'élément cliquable 'déclencheur'.

Attributs HTML (optionnels) :
  • position : définit la position de 'target'.
  • alignment : définit l'alignement de 'target'.
  • rel : lie 'trigger' à un élément 'target' en dehors du conteneur 'anchor'.

Target

Nom CSS : ddkit-target.

L'élément animé (dropdown).

Code HTML

Stucture HTML représentative à utiliser avec Dropdown Kit :

<!-- Ancre ('anchor') -->
<baliseauchoix class="ddkit-anchor">

    <!-- Déclencheur ('trigger') -->
    <baliseauchoix class="ddkit-trigger">l'intitulé du lien</baliseauchoix>

    <!-- Cible ('target') -->
    <baliseauchoix class="ddkit-target">
        le contenu à afficher (liste, image, div, etc.)
    </baliseauchoix>

</baliseauchoix>

Utilisation des attributs HTML et de la classe CSS optionnelle :

<!-- Ancre ('anchor') -->
<baliseauchoix class="ddkit-anchor ddkit-opened">

    <!-- Déclencheur ('trigger') -->
    <baliseauchoix class="ddkit-trigger" rel="#ExternalTarget" position="'top'" alignment="2">l'intitulé du lien</baliseauchoix>

    <!-- Cible ('target') -->
    <baliseauchoix class="ddkit-target">
        le contenu à afficher (liste, image, div, etc.)
    </baliseauchoix>

</baliseauchoix>

CSS

Styles de référence

Les styles de Dropdown Kit se trouvent dans le fichier dropdownkit.css.

Libre à vous de les personnaliser.

Dans la partie <head> de la page HTML :

<!-- Plugin styles -->
<link rel="stylesheet" type="text/css" href="css/dropdownkit.css" media="screen, projection" />

Javascript

Librairie

Le fichier javascript principal est actuellement jquery.dropdownkit.1.0.js.

En option, vous pouvez inclure les plugins suivants :

  • jquery.easing.1.3.min.js, pour les effets de "easing" sur les transitions.
    Plus d'info sur le site : easing plugin homepage

Utilisation

Dans la section <head> de la page HTML :

<!-- jQuery library -->
<script type="text/javascript" src="your_js_path/jquery-1.8.3.min.js"></script>

<!-- jQuery add-ons (optional) -->
<script type="text/javascript" src="your_js_path/jquery.easing.1.3.min.js"></script>

<!-- Dropdown Kit core -->
<script type="text/javascript" src="your_js_path/jquery.dropdownkit.1.0.js"></script>

<!-- Dropdown Kit execution -->
<script type="text/javascript">
    $(window).load(function(){
        
        $(".selecteur").dropdownkit();

    });
</script>

Vous pouvez alors paramétrer et lancer le plugin :

<!-- Dropdown Kit execution -->
<script type="text/javascript">
    $(window).load(function(){
        
        $(".selecteur").dropdownkit({
            bodyclick :false,
            autoclose :false
        });

    });
</script>
Se référer à la rubrique 'Options' ci-dessous.

Méthodes de l'API

setTargetPos ( position, alignment )

Change la position/l'alignement de l'élément 'target'.

openTarget ( fx )

Affiche un élément 'target'.

closeTarget ( fx )

Masque un élément 'target'.

closeAllTargets ()

Ferme tous les éléments 'target'. Si un nom de groupe est défini, la fermeture s'appliquera uniquement à ses membres.

Dépannage

Erreurs fréquentes

Si vous rencontrez des difficultés en utilisant le plugin, commencez par les vérifications basiques :

  • Assurez-vous que les 3 éléments HTML principaux ('anchor', 'trigger' et 'target') sont bien présents dans le code.
  • Si vous utilisez un élément 'target' situé à l'extérieur du bloc 'anchor', assurez-vous que le caractère de sélection ('.' ou '#) est présent devant le nom de classe ou d'ID passé en paramètre.
