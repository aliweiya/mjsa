
# Blink

Chrome 28+ 和 Opera 14+ 会使用 Blink 渲染引擎，但是目前 Chrome 没有在
User-Agent 中修改渲染引擎名称的计划。

## 名称由来

"Blink" is an acronym of "Bruce Lawson in nice knickers".

## 使用情况

直到 Chrome 28.0.1464.0 开发版中还是使用 Webkit。
